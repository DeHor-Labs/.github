# DeHor Labs

Agent infrastructure, MCP servers, and developer tools for software teams that want AI agents to operate real workflows instead of producing throwaway demos.

We build small, sharp systems around three ideas:

- agents need reliable tools, not just prompts;
- context should become searchable operational memory;
- automation is only useful when it survives real CI, browser state, credentials, and production constraints.

## Start Here

### Agent Video Intelligence

- **[TranscreveAI](https://github.com/DeHor-Labs/transcreve-ai)** - turns video links, Reels, captions, frames, OCR, and transcripts into multimodal dossiers, content packs, skill drafts, batch reports, and RAG-ready knowledge.

### Agent Tooling

- **[Visual Eyes](https://github.com/DeHor-Labs/visual-eyes)** - gives coding agents visual inspection for running web apps.
- **[Semtree](https://github.com/DeHor-Labs/semtree)** - semantic code-tree tooling for AI assistants working with large codebases.
- **[agent-stack-public](https://github.com/DeHor-Labs/agent-stack-public)** - public map of a Codex and Claude Code agent stack.

### MCP And Domain Workflows

- **[MCP Fiscal Brasil](https://github.com/DeHor-Labs/mcp-fiscal-brasil)** - MCP server for Brazilian fiscal workflows, including CNPJ, NFe, NFSe, SPED, and eSocial.

### Safety And Orchestration

- **[ShieldCode](https://github.com/DeHor-Labs/shieldcode)** - security hardening and production-grade error-handling guidance for AI-assisted development.
- **[claude-orchestration](https://github.com/DeHor-Labs/claude-orchestration)** - multi-agent orchestration guidelines and reusable operating patterns.

## What We Care About

- multimodal knowledge extraction from messy real-world inputs;
- MCP surfaces that agents can call safely;
- review gates, CI automation, and traceable decisions;
- Brazilian business, fiscal, and operational automation;
- tools that make agents inspect, verify, and remember.

## Signature Project

**TranscreveAI** is the current center of gravity: an agent-ready video intelligence pipeline that can process a Reel, detect source constraints, extract captions and frames, create evidence-backed dossiers, generate content/skill/backlog artifacts, index the result, and hand it back to a parent agent as reusable knowledge.
