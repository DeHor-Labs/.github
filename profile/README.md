# DeHor Labs

DeHor Labs is an independent software lab for practical open-source tools, automation, and applied AI.

This organization is the home for public projects that are useful enough to keep improving: agent tooling, MCP servers, developer utilities, and applied AI products built around real workflows.

## Focus

- Developer tools and workflow automation
- Applied AI products and agents
- Infrastructure glue for small teams and indie builders
- Public experiments with real-world constraints

## Projects

- [Transcreve AI](https://github.com/DeHor-Labs/transcreve-ai) - multimodal video dossiers for knowledge bases, with transcription, OCR, search, and agent-oriented workflows.
- [MCP Fiscal Brasil](https://github.com/DeHor-Labs/mcp-fiscal-brasil) - an MCP server for Brazilian fiscal data such as CNPJ, NFe, NFSe, SPED, and eSocial.
- [Semtree](https://github.com/DeHor-Labs/semtree) - semantic code trees for AI assistants and repository understanding.
- [Visual Eyes](https://github.com/DeHor-Labs/visual-eyes) - visual testing, regression detection, and auto-fix workflows for agentic development.
- [Shieldcode](https://github.com/DeHor-Labs/shieldcode) - security hardening and production-grade error-handling guidance for coding agents.
- [Agent Stack Public](https://github.com/DeHor-Labs/agent-stack-public) - a public map of a Codex and Claude Code agent stack.
- [Claude Orchestration](https://github.com/DeHor-Labs/claude-orchestration) - multi-agent orchestration guidelines for Claude Code.

## Principles

- Useful before clever
- Clear behavior over vague demos
- Simple interfaces for messy workflows
- Built from Brazil, useful anywhere
